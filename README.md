# Plant Twin

Web-based 3D factory layout and digital twin editor for Siemens Plant Simulation-like planning.

## Structure

- React handles the web UI, panels, lists, and application state.
- Vanilla JavaScript + Three.js handles the 3D scene, camera, selection, and rendering.
- React talks to the Three.js engine through a narrow imperative API.

## Scripts

- `npm run dev`
- `npm run build`
- `npm run preview`
